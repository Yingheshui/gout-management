# 高尿酸·痛风管理指南

这是一个专门为高尿酸和痛风患者设计的健康管理网页，帮助患者更好地控制饮食、生活习惯，达到降低尿酸的效果。

## 功能特点

- 📊 详细的尿酸参考值和控制目标
- 🍽️ 食物嘌呤分级管理（高、中、低嘌呤食物）
- 📋 常见食物嘌呤含量对照表
- 💧 饮水建议和推荐饮品
- 🏃 生活方式管理指南
- 🚨 急性痛风发作应对措施
- ✅ 每日健康清单
- ⏰ 定期检查项目提醒

## 使用说明

本网页可直接在浏览器中打开使用，所有内容均基于医学指南和临床实践经验整理。

**重要提示：** 本指南仅供健康教育参考，不能替代医生的诊断和治疗建议。如有任何疑问或症状，请及时就医咨询专业医生。

## 发布到 GitHub Pages

1. 在 GitHub 创建新仓库
2. 推送代码到仓库
3. 在仓库设置中启用 GitHub Pages
4. 选择 main/master 分支作为发布源

## 许可

本项目仅用于健康教育目的。
